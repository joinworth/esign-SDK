ESIGN SDK Development Example

Developer Mode Implementation

<esign-component api-key="dev-api-key-123" document-id="DOC001" dev-mode ></esign-component>
Note: This example runs in developer mode, where the signing process is mocked and no actual API calls are made. Check the browser console to see the mock API interactions.

Development Testing Guide

  1. Include the ESIGN SDK in your HTML file
  2. Add the <esign-component> element with test credentials
  3. Always include dev-mode attribute during development
  4. Click "Start Signing" to test the mock signing flow
  5. Check browser console for mock API interactions
